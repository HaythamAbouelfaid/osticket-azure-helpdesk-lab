# Install Guide (High Level)

1. Provision Azure VM (Ubuntu)
2. Install Apache + PHP dependencies
3. Install MySQL and create osTicket database/user
4. Deploy osTicket into Apache web root
5. Complete web installer (DB + admin)
6. Secure with HTTPS using Certbot
7. Configure:
   - Departments
   - Help Topics
   - Agents/Roles
   - SLA Plans
8. Run ticket simulation and record metrics
